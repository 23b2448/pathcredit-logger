# pathcredit-logger

## How to run

1. Download or clone the repository
2. Open index.html in any browser and no setup required.

## What I built

- A simple web app to log student activities
- Form with: Activity name, Category (Academic, Technical, Cultural, Sports), Date
- Activity feed showing all entries
- Filter to view activities by category
- Updates happen instantly without page reload

## Edge cases handled

- Prevented submission with empty fields
- Show message when no activities are logged
- Show message when no results match selected filter
